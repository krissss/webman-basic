# 项目概述

## 简介

这是一个基于 [Webman](https://www.workerman.net/webman) 框架构建的基础项目模板，旨在提供一个高性能、易于使用的 PHP
应用开发基础。项目集成了多种常用组件和技术，适合构建 Web 应用、API 接口、后台管理系统等。

## 技术栈

- **核心框架**: [Webman 2.x](https://www.workerman.net/webman)
- **PHP 版本**: >= 8.2
- **数据库迁移**: [Phinx](https://phinx.org/)
- **ORM**: Laravel Eloquent
- **后台管理**: [Amis](https://aisuda.bce.baidu.com/amis/zh-CN/docs/index) 可视化页面构建工具
- **API 文档**: [Swagger/OpenAPI](https://swagger.io/specification/)
- **测试框架**: [PestPHP](https://pestphp.com/)
- **代码质量**: [PHPStan](https://phpstan.org/)
- **国际化**: Symfony Translation / Laravel Translation

## 模块说明

- **admin模块**: 提供后台管理界面，基于Amis实现可视化操作界面
- **api模块**: 提供RESTful API接口，并集成Swagger文档
- **components模块**: 封装了项目通用的基础组件
- **enums模块**: 定义项目中使用的枚举类型
- **model模块**: 数据模型，基于Laravel Eloquent ORM
